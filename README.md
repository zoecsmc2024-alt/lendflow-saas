🚀 LendFlow Africa (SaaS)
The Digital Operating System for Modern Tier-4 Lenders.

LendFlow Africa is a multi-tenant FinTech platform designed to digitize the lending lifecycle for microfinance institutions. Built for scale, security, and speed.

🏗️ Core Pillars
Dashboard: Real-time CEO analytics and portfolio health.

Portfolio: KYC-compliant borrower management and loan disbursement.

Treasury: Integrated tracking for payments, expenses, and petty cash.

Admin: Role-Based Access Control (RBAC) and automated payroll.

Settings: Dynamic tenant branding and custom lending rules.

🛠️ Tech Stack
Frontend: Streamlit (Python)

Database: Supabase (PostgreSQL)

Auth: Supabase GoTrue (JWT-based)

Communications: WhatsApp API Integration

🔐 Security & Privacy
Data Isolation: Row-Level Security (RLS) ensures no cross-tenant data leakage.

Encryption: All sensitive financial records and passwords are encrypted at rest.
